# Daniel Diaz - Textile Design Portfolio

A high-fidelity portfolio website featuring horizontal scrolling and interactive pattern reveals.

## Tech Stack
- **Framework**: Vite + React
- **Styling**: Tailwind CSS
- **Animation**: Framer Motion
- **Scroll**: Lenis

## Features
- **Horizontal Scroll**: A seamless horizontal navigation experience.
- **Pattern Reveal**: Hover over textile patterns to reveal the fashion mockup underneath.
- **Interactive**: Smooth parallax and cursor-based masking.

## Setup

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Locally**:
   ```bash
   npm run dev
   ```

3. **Build**:
   ```bash
   npm run build
   ```

## Customization

### Adding Projects
Edit `src/App.jsx` and add new entries to the `projects` array.

### Changing Assets
Replace images in `src/assets/`.
- `pattern1.png`: Top layer (Textile)
- `mockup1.png`: Bottom layer (Fashion Mockup)

Ensure images are high-resolution for the best experience.
